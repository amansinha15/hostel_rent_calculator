# Hostel Rent Calculator 🏠💰

A beginner-friendly Python project that helps calculate and split hostel or flat expenses among roommates. This program takes rent, food expenses, electricity usage, and the number of people living together, then calculates how much each person needs to pay.

---

## 🚀 Features

- Calculate hostel/flat rent easily
- Add food/snack expenses
- Include electricity bill calculation
- Split total expenses equally among roommates
- Simple command-line interface
- Beginner-friendly Python project

---

## 🛠️ Built With

- Python 3
- VS Code

---

## 📂 Project Overview

The program takes:

- Total rent
- Food expenses
- Electricity units consumed
- Charge per electricity unit
- Number of persons living in the room/flat

Then it calculates the total amount each person has to pay.

---

## ▶️ How to Run the Project

1. Clone this repository:

```bash
git clone https://github.com/your-username/Hostel-Rent-Calculator.git
```

2. Open the project folder:

```bash
cd Hostel-Rent-Calculator
```

3. Run the Python file:

```bash
python Hostel_Rent_Calculator.py
```
